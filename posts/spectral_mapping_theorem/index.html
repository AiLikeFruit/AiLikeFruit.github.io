谱映射定理 (Spectral Mapping Theorem)

1. 定理核心陈述

谱映射定理描述了矩阵变换与特征值变换之间的同步关系。简单来说：如果你对矩阵 AAA 施加某种函数变换（如多项式运算），那么新矩阵的特征值，就是原特征值经过同种函数运算后的结果。

数学表达

设 AAA 是一个 n×nn \times nn×n 矩阵，λ\lambdaλ 是 AAA 的一个特征值。如果 P(x)P(x)P(x) 是一个多项式，那么：

P(λ) 是矩阵 P(A) 的特征值P(\lambda) \text{ 是矩阵 } P(A) \text{ 的特征值}P(λ) 是矩阵 P(A) 的特征值

更形式化地，如果记 σ(A)\sigma(A)σ(A) 为矩阵 AAA 的谱（所有特征值的集合），则有：

σ(P(A))=P(σ(A))={P(λ)∣λ∈σ(A)} \sigma(P(A)) = P(\sigma(A)) = \{ P(\lambda) \mid \lambda \in \sigma(A) \} σ(P(A))=P(σ(A))={P(λ)∣λ∈σ(A)}

2. 推导证明 (多项式情况)

利用特征值的定义，可以轻松证明这一点。

  1. 定义：设 λ\lambdaλ 是 AAA 的特征值，xxx 是对应的非零特征向量。 Ax=λxAx = \lambda xAx=λx
  2. 幂运算：矩阵的幂次对应特征值的幂次。 Akx=λkxA^k x = \lambda^k xAkx=λkx
  3. 多项式运算：设 P(t)=antn+⋯+a1t+a0P(t) = a_n t^n + \dots + a_1 t + a_0P(t)=an​tn+⋯+a1​t+a0​。 P(A)x=(anAn+⋯+a1A+a0I)x=an(Anx)+⋯+a1(Ax)+a0(Ix)=an(λnx)+⋯+a1(λx)+a0x=(anλn+⋯+a1λ+a0)x=P(λ)x \begin{aligned} P(A)x &= (a_n A^n + \dots + a_1 A + a_0 I)x \\ &= a_n(A^n x) + \dots + a_1(Ax) + a_0(Ix) \\ &= a_n(\lambda^n x) + \dots + a_1(\lambda x) + a_0 x \\ &= (a_n \lambda^n + \dots + a_1 \lambda + a_0) x \\ &= P(\lambda) x \end{aligned} P(A)x​=(an​An+⋯+a1​A+a0​I)x=an​(Anx)+⋯+a1​(Ax)+a0​(Ix)=an​(λnx)+⋯+a1​(λx)+a0​x=(an​λn+⋯+a1​λ+a0​)x=P(λ)x​
  4. 结论： 由于 P(A)x=P(λ)xP(A)x = P(\lambda)xP(A)x=P(λ)x 且 x≠0x \neq 0x=0，根据特征值定义，P(λ)P(\lambda)P(λ) 即为 P(A)P(A)P(A) 的特征值，且对应的特征向量依然是 xxx。

3. 与“零化多项式”的关系

这是理解“为什么零化多项式能推出特征值为根”的关键，它是谱映射定理的一个特例。

  • 场景：假设 P(x)P(x)P(x) 是 AAA 的零化多项式。
  • 条件：这意味着 P(A)=0P(A) = \mathbf{0}P(A)=0（零矩阵）。
  • 应用定理：根据谱映射定理，P(A)P(A)P(A) 的特征值应该是 P(λ)P(\lambda)P(λ)。
  • 推导： 零矩阵 0\mathbf{0}0 的特征值全都是 000。 ∴P(λ)=0\therefore P(\lambda) = 0∴P(λ)=0
  • 结论：矩阵 AAA 的特征值 λ\lambdaλ 必定是方程 P(x)=0P(x)=0P(x)=0 的根。

4. 推广：矩阵函数

该定理不仅适用于多项式，还适用于收敛的级数定义的矩阵函数（Matrix Functions）。

如果 f(t)f(t)f(t) 是在 AAA 的谱上定义的解析函数（例如 et,sin⁡t,cos⁡te^t, \sin t, \cos tet,sint,cost 等），那么：

若 λ∈σ(A)  ⟹  f(λ)∈σ(f(A))\text{若 } \lambda \in \sigma(A) \implies f(\lambda) \in \sigma(f(A))若 λ∈σ(A)⟹f(λ)∈σ(f(A))

常见例子

  • 矩阵指数：eAe^AeA 的特征值是 eλe^\lambdaeλ。
  • 逆矩阵：如果 AAA 可逆（λ≠0\lambda \neq 0λ=0），则 A−1A^{-1}A−1 的特征值是 λ−1\lambda^{-1}λ−1（此时函数为 f(t)=1/tf(t) = 1/tf(t)=1/t）。
  • 平移：矩阵 A−3IA - 3IA−3I 的特征值是 λ−3\lambda - 3λ−3。

一句话记忆：矩阵怎么变，特征值就跟着怎么变；如果矩阵变成了 0，特征值算出来也得是 0。

一阶常系数齐次线性微分方程解的证明

1. 问题定义

我们探讨的目标方程如下：

dxdt=ax \frac{dx}{dt} = ax dtdx​=ax
  • x(t)x(t)x(t)：关于时间 ttt 的未知函数。
  • aaa：常数（Constant Coefficient）。
  • 初始条件：x(0)=x0x(0) = x_0x(0)=x0​。

我们需要证明该方程的通解为：x(t)=Ceatx(t) = C e^{at}x(t)=Ceat。

2. 证明方法一：分离变量法 (Separation of Variables)

这是最严谨的推导方法，通过微积分运算直接求出解析解。

步骤 1：变量分离 假设 x≠0x \neq 0x=0，将含有 xxx 的项移至等式左边，含有 ttt 的项移至等式右边：

1x dx=a dt \frac{1}{x} \, dx = a \, dt x1​dx=adt

步骤 2：两边积分 对等式两边同时进行不定积分：

∫1x dx=∫a dt \int \frac{1}{x} \, dx = \int a \, dt ∫x1​dx=∫adt

步骤 3：计算积分结果 左边积分得到对数函数，右边积分得到一次函数（注意积分常数 C1C_1C1​）：

ln⁡∣x∣=at+C1 \ln|x| = at + C_1 ln∣x∣=at+C1​

步骤 4：反解 xxx 为了去掉对数符号，对等式两边取指数 eee：

∣x∣=eat+C1 |x| = e^{at + C_1} ∣x∣=eat+C1​

利用指数运法则 em+n=em⋅ene^{m+n} = e^m \cdot e^nem+n=em⋅en，展开为：

∣x∣=eC1⋅eat |x| = e^{C_1} \cdot e^{at} ∣x∣=eC1​⋅eat

步骤 5：处理常数 去掉绝对值符号。由于 eC1e^{C_1}eC1​ 是一个恒正的常数，我们可以定义一个新的任意常数 CCC（CCC 可以为正、负或 0）：

C=±eC1 C = \pm e^{C_1} C=±eC1​

于是得到通解公式：

x(t)=Ceat x(t) = C e^{at} x(t)=Ceat

步骤 6：确定常数 CCC 代入初始时刻 t=0t=0t=0：

x(0)=C⋅ea⋅0=C⋅1=C x(0) = C \cdot e^{a \cdot 0} = C \cdot 1 = C x(0)=C⋅ea⋅0=C⋅1=C

因此，常数 CCC 即为初始值 x(0)x(0)x(0)。

3. 证明方法二：求导验证法 (Verification)

这种方法基于对指数函数性质的直观理解。

核心思想 方程 dxdt=ax\frac{dx}{dt} = axdtdx​=ax 的物理意义是：函数的导数（变化率）等于常数 aaa 乘以函数本身。

推导 回顾基本求导公式，我们知道自然指数函数 eate^{at}eat 具有唯一的“导数形式不变”的性质：

ddt(eat)=a⋅eat \frac{d}{dt}(e^{at}) = a \cdot e^{at} dtd​(eat)=a⋅eat

验证 假设解的形式为 x(t)=Ceatx(t) = C e^{at}x(t)=Ceat，将其代入原方程检验：

  • 左边 (LHS)： ddt(Ceat)=C⋅ddt(eat)=C⋅(aeat)=a(Ceat) \frac{d}{dt}(C e^{at}) = C \cdot \frac{d}{dt}(e^{at}) = C \cdot (a e^{at}) = a(C e^{at}) dtd​(Ceat)=C⋅dtd​(eat)=C⋅(aeat)=a(Ceat)
  • 右边 (RHS)： a⋅x=a⋅(Ceat) a \cdot x = a \cdot (C e^{at}) a⋅x=a⋅(Ceat)

结论

LHS=RHS \text{LHS} = \text{RHS} LHS=RHS

等式成立，故 x(t)=Ceatx(t) = C e^{at}x(t)=Ceat 确为该方程的解。

这一结论直接推广到了一阶常系数齐次线性微分方程组（即你题目中的情况）：
